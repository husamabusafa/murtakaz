# Arabic Glossary (Saudi Market)

Use these Arabic terms consistently across UI, content, and documentation for the Saudi market.

## Core product terms
- Organization (tenant): الجهة (جمع: الجهات)
- Department: الإدارة (جمع: الإدارات)
- Strategy execution: تنفيذ الاستراتيجية
- Governance: الحوكمة
- Compliance: الامتثال
- Audit trail / audit log: مسار التدقيق / سجل التدقيق
- Approval: اعتماد / الموافقات
- Role-based access control (RBAC): إدارة الوصول حسب الدور (RBAC)

## KPI terms
- KPI: مؤشر أداء رئيسي
- KPIs: مؤشرات الأداء الرئيسية
- KPI performance: أداء مؤشرات الأداء الرئيسية
- KPI approval level: مستوى اعتماد مؤشرات الأداء الرئيسية
- KPI owner: مسؤول مؤشر الأداء الرئيسي
- KPI variance: انحراف مؤشرات الأداء الرئيسية
- KPI freshness: حداثة مؤشرات الأداء الرئيسية

## Portfolio terms
- Pillar: ركيزة
- Initiative: مبادرة
- Project: مشروع
- Risk: مخاطرة
- Escalation: تصعيد

## Roles
- PMO: مكتب إدارة المشاريع (PMO)
- Executive: تنفيذي
- Admin: مسؤول
