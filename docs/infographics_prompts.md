# Infographics Prompts (Nano Pnana Pro)

Use these prompt templates to generate infographic images for the Murtakaz strategy-execution system. Replace anything in `[BRACKETS]` with your details.

## Global Style (apply to all)

**Prompt add-on (paste at the end of any prompt):**

```
Style: clean modern Saudi government/enterprise look, minimal, high legibility.
Palette: [BRAND_COLORS] (primary/secondary/accent), light background, high contrast.
Typography: Arabic-first, clear Naskh/Sans Arabic, correct diacritics when needed, no decorative calligraphy.
Language: Arabic (Saudi business tone) with small English labels only where useful (bilingual UI feel).
Icons: simple line icons, consistent stroke, no emojis.
Layout: grid-based, generous whitespace, RTL-aware alignment for Arabic.
Branding: include [LOGO] in a corner if provided; otherwise leave space for logo.
Export: [SIZE] (e.g., 1920x1080, 1080x1350, A4), 300 DPI if print.
```

## Quick Variations

- **Social (LinkedIn):** set `[SIZE]` to `1080x1350` and increase font sizes.
- **Slide (16:9):** set `[SIZE]` to `1920x1080`, keep margins wide.
- **Print (A4):** set `[SIZE]` to `2480x3508`, 300 DPI, avoid tiny text.
